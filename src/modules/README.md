# Modules

Este diretório organiza a evolução para fronteiras modulares.

- `Auth`: identidade e perfil do cliente.
- `Credito`: solicitações, contratos e parcelas.
- `AcordoBoleto`: acordos, parcelas de acordo e emissão de boleto.

No estágio atual, os módulos ainda convivem no mesmo serviço da API, mas o diretório já define a estrutura de migração gradual.

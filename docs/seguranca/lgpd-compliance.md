# LGPD Compliance

## Mapa de Dados Pessoais (resumo)

- **Cadastro:** nome, CPF/CNPJ, email, telefone, endereco.
- **Credito:** renda, historico de pagamento, score, parecer.
- **Cobranca:** acordos, boletos, historico de inadimplencia.
- **Seguranca:** IP, logs de acesso, trilha de auditoria.

## Bases Legais (resumo)

- Execucao de contrato.
- Protecao do credito.
- Cumprimento de obrigacao legal/regulatoria.
- Consentimento (quando aplicavel, especialmente comunicacoes nao essenciais).

## DSAR (Direitos do Titular)

1. Receber solicitacao e protocolar.
2. Validar identidade do titular.
3. Classificar pedido (acesso, retificacao, eliminacao, portabilidade).
4. Orquestrar resposta com donos de dados.
5. Responder dentro do SLA e registrar evidencias.

## Controles Tecnicos Minimos

- Criptografia em transito e repouso.
- Mascaramento de dados sensiveis em UI e logs.
- Trilha de auditoria para acessos e alteracoes.
- Retencao e descarte com politica formal.

## Evidencias para Auditoria

- Inventario de dados (ROPA).
- Registro de consentimento versionado.
- Registro de atendimento DSAR.
- Registros de descarte e legal hold.
- Logs de seguranca e acessos administrativos.

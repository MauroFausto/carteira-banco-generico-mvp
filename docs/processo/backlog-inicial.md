# Backlog Inicial (Execucao Unica)

## Prioridade P0 - Fundacao e Seguranca

- **US-001 (Agente 2):** Como usuario, quero autenticar via Microsoft Identity para acessar o sistema.
- **US-002 (Agente 2):** Como gestor, quero autorizacao por role para operar apenas recursos permitidos.
- **US-003 (Agente 5):** Como time, quero controles OWASP aplicados para reduzir risco de seguranca.
- **US-004 (Agente 4):** Como empresa, quero diretrizes LGPD para dados pessoais e auditoria.

## Prioridade P1 - Credito e Decisao

- **US-101 (Agente 3):** Como cliente, quero solicitar credito informando valor e prazo.
- **US-102 (Agente 3):** Como sistema, quero aplicar regras de avaliacao para sugerir aprovacao/negacao.
- **US-103 (Agente 2/3):** Como supervisor, quero aprovar/negar com parecer registrado.

## Prioridade P1 - Divida, Acordo e Boleto

- **US-201 (Agente 3):** Como sistema, quero calcular saldo devedor com rastreabilidade.
- **US-202 (Agente 3):** Como supervisor, quero gerar acordo com condicoes negociadas.
- **US-203 (Agente 6):** Como cliente, quero baixar boleto em PDF com seguranca.

## Prioridade P2 - Arquitetura e Processo

- **US-301 (Agente 1):** Como time, quero arquitetura de monolito modular documentada para manter coesao.
- **US-302 (Agente 7):** Como time, quero padrao TDD/XP definido para garantir qualidade continua.
- **US-303 (Agente 1/7):** Como time, quero criterios de aceite e contratos tecnicos alinhados.
